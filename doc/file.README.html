Agouti {Build Status] {Code Climate] {Gem Version]

Gem for testing above the fold render on the first tcp round trip.

This gem is a Rack middleware that truncates the gzipped response to 14kb.

Useful for testing critical rendering path optimization.

Installation

Add this line to your application's Gemfile:

gem 'agouti'

And then execute:

$ bundle

Or install it yourself as:

$ gem install agouti

Usage

To enable the middleware, it is necessary to add the following header to the request: <code> X-Agouti-Enable: 1 </code>

It is possible to customize the length of the content that the server will respond with the following header: <code> X-Agouti-Limit: 14000 </code>

Example usage with Cucumber, Capybara and Poltergeist

Given /^(?:|I )navigate to '(.+)'$/ do |page_path|
  visit page_path

Given /^(?:|I )navigate to '(.+)' waiting only one tcp round trip$/ do |page_path|
  page.driver.add_header("X-Agouti-Enable", "1", permanent: false)
  visit page_path
end
